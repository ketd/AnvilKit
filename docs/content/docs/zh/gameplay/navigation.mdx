导航模块提供基于三角形邻接图的 A* 网格寻路，以及用于自动移动的代理转向系统。

## NavMesh

`NavMesh` 是一个 ECS `Resource`，以三角形和自动计算的邻接关系存储可行走表面。

### 构建

```rust
use anvilkit_ecs::navigation::NavMesh;
use glam::Vec3;

let vertices = vec![
    Vec3::new(0.0, 0.0, 0.0),
    Vec3::new(10.0, 0.0, 0.0),
    Vec3::new(10.0, 0.0, 10.0),
    Vec3::new(0.0, 0.0, 10.0),
];
let triangles = vec![
    [0, 1, 2],
    [0, 2, 3],
];

// 邻接关系根据共享边自动计算
let navmesh = NavMesh::new(vertices, triangles);
```

### API

| 方法 | 签名 | 说明 |
|--------|-----------|-------------|
| `new` | `(Vec<Vec3>, Vec<[u32; 3]>) -> Self` | 构建带自动邻接的 NavMesh |
| `find_triangle` | `(&self, Vec3) -> Option<usize>` | 查找包含某点的三角形（XZ 投影） |
| `triangle_centroid` | `(&self, usize) -> Vec3` | 获取三角形的质心 |
| `find_path` | `(&self, Vec3, Vec3) -> Option<Vec<Vec3>>` | A* 寻路，返回路径点 |
| `point_in_triangle` | `(Vec2, Vec2, Vec2, Vec2) -> bool` | 重心坐标点在三角形内测试 |

## A* 寻路

`find_path(start, goal)` 在三角形邻接图上运行 A* 算法，以质心为节点、欧几里得距离为启发函数。返回路径点列表：`[start, centroid_1, centroid_2, ..., goal]`。

```rust
if let Some(path) = navmesh.find_path(
    Vec3::new(1.0, 0.0, 1.0),  // 起点
    Vec3::new(8.0, 0.0, 8.0),  // 目标
) {
    for waypoint in &path {
        println!("  -> {}", waypoint);
    }
}
```

## NavAgent

`NavAgent` 是一个 `Component`，每帧沿计算出的路径移动实体。

| 字段 | 类型 | 默认值 | 说明 |
|-------|------|---------|-------------|
| `speed` | `f32` | — | 移动速度（单位/秒） |
| `path` | `Vec<Vec3>` | `[]` | 当前路径点 |
| `current_waypoint` | `usize` | `0` | 路径中的索引 |
| `arrival_threshold` | `f32` | `0.3` | 视为到达路径点的距离 |
| `enabled` | `bool` | `true` | 代理是否处于激活状态 |

| 方法 | 签名 | 说明 |
|--------|-----------|-------------|
| `new` | `(speed: f32) -> Self` | 创建新的代理 |
| `set_destination` | `(&mut self, &NavMesh, Vec3, Vec3) -> bool` | 计算路径，无路径时返回 false |
| `has_arrived` | `(&self) -> bool` | 到达路径终点时返回 true |

```rust
use anvilkit_ecs::navigation::{NavAgent, NavMesh};

// 生成带 NavAgent 的 NPC
let mut agent = NavAgent::new(5.0); // 5 单位/秒
agent.set_destination(&navmesh, npc_pos, target_pos);
commands.spawn((Transform::from_translation(npc_pos), agent));
```

## NavigationPlugin

`NavigationPlugin` 在 `Update` 中注册 `nav_agent_steering_system`。该系统使用 `DeltaTime` 沿路径移动所有 `NavAgent` 实体。

```rust
app.add_plugins(NavigationPlugin);
app.insert_resource(navmesh);
```
