每个 AnvilKit crate 的自动生成 API 文档可在 [docs.rs](https://docs.rs) 上查阅。这些页面直接从源代码和文档注释构建，提供完整的类型签名、trait 实现和示例。

## Crate 文档

| Crate | 说明 | docs.rs |
|-------|-------------|---------|
| `anvilkit-core` | 核心类型 -- Transform、Time、Timer、Persistence | [docs.rs/anvilkit-core](https://docs.rs/anvilkit-core) |
| `anvilkit-ecs` | 实体组件系统 -- App、Plugin、FixedUpdate、Events、GameState、SerializableRegistry、AutoPlugins | [docs.rs/anvilkit-ecs](https://docs.rs/anvilkit-ecs) |
| `anvilkit-render` | 渲染 -- PBR 管线、StandardMaterial、PostProcessSettings、阴影系统、DemoApp | [docs.rs/anvilkit-render](https://docs.rs/anvilkit-render) |
| `anvilkit-assets` | 资产管线 -- glTF 加载、动画、MeshData 工厂、AudioAsset、热重载 | [docs.rs/anvilkit-assets](https://docs.rs/anvilkit-assets) |
| `anvilkit-audio` | 音频 -- AudioSource、AudioBus、空间音频、循环/变调 | [docs.rs/anvilkit-audio](https://docs.rs/anvilkit-audio) |
| `anvilkit-input` | 输入 -- 键盘、鼠标、手柄、ActionMap、ActionId、AxisBinding | [docs.rs/anvilkit-input](https://docs.rs/anvilkit-input) |

## 本地构建文档

在本地生成并查看 API 文档：

```bash
cargo doc --workspace --no-deps --open
```

此命令为工作区中所有 crate 构建文档，并在浏览器中打开。
