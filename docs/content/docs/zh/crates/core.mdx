`anvilkit-core` 是引擎的基础类型库，为所有其他 crate 提供共享的基础设施——数学类型、时间追踪和持久化。

## Transform

`Transform` 表示 3D 空间中的位置、旋转和缩放（40 字节），基于 `glam` 构建。

### 字段

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `translation` | `Vec3` | `Vec3::ZERO` | 空间位置 |
| `rotation` | `Quat` | `Quat::IDENTITY` | 单位四元数表示的朝向 |
| `scale` | `Vec3` | `Vec3::ONE` | 各轴缩放因子 |

### 构造函数

| 方法 | 签名 | 说明 |
|------|------|------|
| `IDENTITY` | `const` | 零平移、单位旋转、单位缩放 |
| `new` | `(Vec3, Quat, Vec3) -> Self` | 完整构造 |
| `from_translation` | `(Vec3) -> Self` | 仅位置 |
| `from_rotation` | `(Quat) -> Self` | 仅旋转 |
| `from_scale` | `(Vec3) -> Self` | 仅缩放 |
| `from_xyz` | `(f32, f32, f32) -> Self` | 从 x/y/z 创建位置 |
| `from_xy` | `(f32, f32) -> Self` | 2D 位置（z = 0） |
| `from_matrix` | `(Mat4) -> Self` | 从 4x4 矩阵分解 |
| `looking_at` | `(eye: Vec3, target: Vec3, up: Vec3) -> Result<Self>` | 摄像机朝向 |

### Builder 方法（可链式调用）

| 方法 | 签名 |
|------|------|
| `with_translation` | `(self, Vec3) -> Self` |
| `with_rotation` | `(self, Quat) -> Self` |
| `with_scale` | `(self, Vec3) -> Self` |

### 操作

| 方法 | 签名 | 说明 |
|------|------|------|
| `compute_matrix` | `(&self) -> Mat4` | 构建 缩放-旋转-平移 矩阵 |
| `transform_point` | `(&self, Vec3) -> Vec3` | 将完整变换应用到点 |
| `transform_vector` | `(&self, Vec3) -> Vec3` | 仅应用旋转+缩放（忽略平移） |
| `mul_transform` | `(&self, &Transform) -> Transform` | 组合变换 `self * other` |
| `inverse` | `(&self) -> Result<Transform>` | 计算逆变换（缩放轴为零时失败） |
| `is_finite` | `(&self) -> bool` | 所有字段均不含 NaN/Inf 则返回 true |

```rust
use anvilkit_core::math::Transform;
use glam::{Vec3, Quat};

// 创建并链式设置
let t = Transform::from_xyz(1.0, 2.0, 3.0)
    .with_rotation(Quat::from_rotation_y(std::f32::consts::FRAC_PI_4))
    .with_scale(Vec3::splat(2.0));

// 组合 parent * child
let parent = Transform::from_xyz(10.0, 0.0, 0.0);
let child  = Transform::from_xyz(5.0, 0.0, 0.0);
let world  = parent.mul_transform(&child);
assert!((world.translation - Vec3::new(15.0, 0.0, 0.0)).length() < 1e-5);

// 逆变换
let inv = t.inverse().unwrap();
let identity = t.mul_transform(&inv);
assert!(identity.translation.length() < 1e-4);
```

## GlobalTransform

`GlobalTransform` 包装一个 `Mat4`，表示最终的世界空间变换。
通常由 `anvilkit-ecs` 中的变换传播系统自动计算。

### API

| 方法 | 签名 | 说明 |
|------|------|------|
| `IDENTITY` | `const` | 单位矩阵 |
| `from_matrix` | `(Mat4) -> Self` | 包装已有矩阵 |
| `from_transform` | `(&Transform) -> Self` | 从本地 Transform 转换 |
| `matrix` | `(&self) -> Mat4` | 获取原始矩阵 |
| `translation` | `(&self) -> Vec3` | 提取平移分量 |
| `rotation` | `(&self) -> Quat` | 分解并返回旋转 |
| `scale` | `(&self) -> Vec3` | 分解并返回缩放 |
| `transform_point` | `(&self, Vec3) -> Vec3` | 变换点 |
| `transform_vector` | `(&self, Vec3) -> Vec3` | 变换向量（无平移） |
| `mul_transform` | `(&self, &GlobalTransform) -> GlobalTransform` | 矩阵乘法 |
| `inverse` | `(&self) -> Result<Self>` | 矩阵求逆（奇异矩阵时失败） |
| `is_finite` | `(&self) -> bool` | 所有元素均有限则返回 true |

`GlobalTransform` 实现了 `From<Transform>` 和 `From<Mat4>`。

```rust
use anvilkit_core::math::{Transform, GlobalTransform};
use glam::Vec3;

let local = Transform::from_xyz(5.0, 10.0, 15.0);
let global: GlobalTransform = local.into();
assert_eq!(global.translation(), Vec3::new(5.0, 10.0, 15.0));
```

## Time

`Time` 追踪帧间隔、总运行时间和帧计数。通常作为 ECS 资源存储，每帧更新一次。

### API

| 方法 | 签名 | 说明 |
|------|------|------|
| `new` | `() -> Self` | 以当前时刻作为启动时间创建 |
| `update` | `(&mut self)` | 前进到当前时刻（每帧调用一次） |
| `delta` | `(&self) -> Duration` | 上次 `update()` 以来的时间 |
| `delta_seconds` | `(&self) -> f32` | Delta 的 `f32` 秒数 |
| `delta_seconds_f64` | `(&self) -> f64` | Delta 的 `f64` 秒数 |
| `delta_millis` | `(&self) -> u128` | Delta 的毫秒数 |
| `elapsed` | `(&self) -> Duration` | 从启动以来的时间 |
| `elapsed_seconds` | `(&self) -> f32` | 已过时间的 `f32` 秒数 |
| `frame_count` | `(&self) -> u64` | 已处理的总帧数 |
| `fps` | `(&self) -> f64` | 平均 FPS（帧数 / 已过时间） |
| `instant_fps` | `(&self) -> f64` | 瞬时 FPS（1 / delta） |
| `is_first_frame` | `(&self) -> bool` | 在首次 `update()` 之前返回 true |
| `reset` | `(&mut self)` | 将所有计数器归零 |
| `with_scale` | `(&self, f32) -> ScaledTime` | 创建时间缩放包装器 |

```rust
use anvilkit_core::time::Time;

let mut time = Time::new();

// 游戏循环
time.update();
let dt = time.delta_seconds();   // 距上次更新的秒数
let total = time.elapsed_seconds();
println!("帧 {} | dt={:.4}s | FPS={:.1}", time.frame_count(), dt, time.fps());

// 慢动作
let slow = time.with_scale(0.5);
let slow_dt = slow.delta_seconds(); // dt * 0.5
```

## Timer

`Timer` 提供一次性和重复倒计时器，支持暂停/恢复。

### 构造函数

| 方法 | 说明 |
|------|------|
| `new(Duration)` | 一次性计时器 |
| `repeating(Duration)` | 重复计时器 |
| `from_seconds(f32)` | 一次性（秒） |
| `repeating_from_seconds(f32)` | 重复（秒） |
| `from_millis(u64)` | 一次性（毫秒） |
| `repeating_from_millis(u64)` | 重复（毫秒） |

### 核心 API

| 方法 | 签名 | 说明 |
|------|------|------|
| `tick` | `(&mut self, Duration)` | 以 delta 推进计时器 |
| `finished` | `(&self) -> bool` | elapsed >= duration 时为 true |
| `just_finished` | `(&self) -> bool` | 仅在计时器完成的那一帧为 true |
| `percent` | `(&self) -> f32` | 进度 0.0..1.0 |
| `elapsed` | `(&self) -> Duration` | 已经过的时间 |
| `remaining` | `(&self) -> Duration` | 剩余时间 |
| `reset` | `(&mut self)` | 重置为零，状态改为 Running |
| `pause` | `(&mut self)` | 暂停（tick 变为空操作） |
| `resume` | `(&mut self)` | 从暂停状态恢复 |
| `finish` | `(&mut self)` | 立即强制完成 |
| `set_duration` | `(&mut self, Duration)` | 更改目标时长 |
| `set_repeating` | `(&mut self, bool)` | 切换重复模式 |
| `is_running` / `is_paused` | `(&self) -> bool` | 状态查询 |

```rust
use anvilkit_core::time::Timer;
use std::time::Duration;

// 武器冷却：0.5 秒重复
let mut cooldown = Timer::repeating_from_seconds(0.5);

// 每帧
cooldown.tick(Duration::from_millis(16));
if cooldown.just_finished() {
    // 可以再次开火
}

// 延迟事件：3 秒一次性
let mut delay = Timer::from_seconds(3.0);
delay.tick(delta);
if delay.just_finished() {
    // 触发过场动画
}
```

## 持久化（feature `"persistence"`）

持久化模块提供三层：玩家 **Settings**、多槽位 **SaveManager** 和原始键值 **WorldStorage**。

### Settings

以 RON 格式存储的类型化配置。文件缺失或损坏时回退到默认值。

#### 子结构体

**GraphicsSettings**

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `width` | `u32` | `1280` | 窗口宽度 |
| `height` | `u32` | `720` | 窗口高度 |
| `fullscreen` | `bool` | `false` | 全屏模式 |
| `vsync` | `bool` | `true` | 垂直同步 |
| `msaa` | `u32` | `4` | MSAA 采样数（1/2/4） |
| `bloom` | `bool` | `true` | 泛光后处理 |
| `ssao` | `bool` | `true` | 屏幕空间环境光遮蔽 |
| `shadow_quality` | `u32` | `2` | 0=关、1=低、2=中、3=高 |

**AudioSettings**

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `master_volume` | `f32` | `1.0` | 主音量 0.0..1.0 |
| `music_volume` | `f32` | `0.8` | 音乐音量 0.0..1.0 |
| `sfx_volume` | `f32` | `1.0` | 音效音量 0.0..1.0 |

**InputSettings**

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `mouse_sensitivity` | `f32` | `0.003` | 鼠标灵敏度倍率 |
| `invert_y` | `bool` | `false` | 反转 Y 轴 |
| `action_overrides` | `HashMap<String, String>` | `{}` | 自定义按键重绑定 |

#### Settings API

| 方法 | 签名 | 说明 |
|------|------|------|
| `load` | `(path: &Path) -> Self` | 从 RON 加载；出错时返回默认值 |
| `save` | `(&self, path: &Path) -> Result<(), String>` | 写入 RON（自动创建目录） |
| `default_path` | `() -> PathBuf` | `"config/settings.ron"` |

```rust
use anvilkit_core::persistence::Settings;

// 加载（或使用默认值）
let mut settings = Settings::load(&Settings::default_path());
settings.graphics.vsync = false;
settings.audio.master_volume = 0.7;
settings.save(&Settings::default_path()).unwrap();
```

### SaveManager

管理多个存档槽位，每个槽位包含元数据（`meta.ron`）和用于 `WorldStorage` 的 `data/` 目录。

#### 目录布局

```
saves/
  quick/
    meta.ron
    data/        <-- WorldStorage 根目录
  slot_1/
    meta.ron
    data/
```

#### SaveSlotInfo 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `name` | `String` | 槽位名（如 `"quick"`、`"slot_1"`） |
| `timestamp` | `u64` | Unix 时间戳（秒） |
| `play_time_secs` | `f64` | 累计游玩时长 |
| `game_version` | `String` | 游戏版本字符串 |
| `metadata` | `HashMap<String, String>` | 游戏特定元数据 |

#### SaveManager API

| 方法 | 签名 | 说明 |
|------|------|------|
| `new` | `(dir: impl AsRef<Path>, version: &str) -> Result<Self, String>` | 创建/打开存档目录 |
| `save` | `(&self, slot: &str, play_time: f64, metadata: HashMap) -> Result<PathBuf, String>` | 写入元数据，返回 `data/` 路径 |
| `list_saves` | `(&self) -> Vec<SaveSlotInfo>` | 所有槽位按最新优先排序 |
| `get_save_info` | `(&self, slot: &str) -> Option<SaveSlotInfo>` | 读取单个槽位的元数据 |
| `delete` | `(&self, slot: &str) -> Result<(), String>` | 递归删除槽位目录 |
| `slot_data_path` | `(&self, slot: &str) -> PathBuf` | 获取 `data/` 路径（不写入元数据） |

```rust
use anvilkit_core::persistence::SaveManager;
use std::collections::HashMap;

let mgr = SaveManager::new("saves", "1.0.0").unwrap();

// 创建存档
let mut meta = HashMap::new();
meta.insert("level".into(), "3".into());
let data_path = mgr.save("slot_1", 3600.0, meta).unwrap();

// 列出存档
for info in mgr.list_saves() {
    println!("{}: 游玩 {}s", info.name, info.play_time_secs);
}

// 删除
mgr.delete("slot_1").unwrap();
```

### WorldStorage

基于文件系统的键值存储。每个 key 映射到基目录下的一个文件。key 中的斜杠会创建子目录。写入具有原子性（先写 `.tmp` 再 rename）。

#### API

| 方法 | 签名 | 说明 |
|------|------|------|
| `open` | `(path: impl AsRef<Path>) -> Result<Self, String>` | 打开/创建存储目录 |
| `get` | `(&self, key: &str) -> Option<Vec<u8>>` | 读取值；不存在返回 `None` |
| `put` | `(&self, key: &str, value: &[u8]) -> Result<(), String>` | 原子写入（tmp + rename） |
| `delete` | `(&self, key: &str) -> Result<(), String>` | 删除键 |
| `keys_with_prefix` | `(&self, prefix: &str) -> Vec<String>` | 列出以 prefix 开头的所有键 |
| `batch_put` | `(&self, entries: &[(&str, &[u8])]) -> Result<(), String>` | 批量写入（每个键原子） |
| `path` | `(&self) -> &Path` | 基目录 |

```rust
use anvilkit_core::persistence::WorldStorage;

let storage = WorldStorage::open("saves/slot_1/data").unwrap();

// 存储区块数据
storage.put("chunk/0/0", &bincode::serialize(&chunk).unwrap()).unwrap();

// 读回
if let Some(data) = storage.get("chunk/0/0") {
    let chunk: Chunk = bincode::deserialize(&data).unwrap();
}

// 枚举所有区块
let keys = storage.keys_with_prefix("chunk/");
println!("已存储 {} 个区块", keys.len());

// 批量写入
storage.batch_put(&[
    ("meta/version", b"1"),
    ("meta/seed", b"42"),
]).unwrap();
```
