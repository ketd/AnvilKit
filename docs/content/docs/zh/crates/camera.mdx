`anvilkit-camera` 提供灵活的相机控制系统。

## CameraController

```rust
use anvilkit_camera::controller::CameraController;

let controller = CameraController {
    mode: CameraMode::FirstPerson,
    move_speed: 10.0,
    mouse_sensitivity: 0.003,
    base_fov: 70.0, // 基准视场角（度）
    ..Default::default()
};
```

内置的 `camera_controller_system` 自动从 `InputState::mouse_delta()` 读取鼠标移动量，引擎会自动将 `DeviceEvent::MouseMotion` 转发到该字段，无需手动管理。

## 相机模式

| 模式 | 说明 |
|------|------|
| `FirstPerson` | 第一人称——相机跟随玩家视角 |
| `ThirdPerson` | 第三人称——相机跟随目标，带偏移 |
| `Free` | 自由飞行——无约束 6DOF 移动 |

## 相机特效

`CameraEffects` 提供头部摇晃、屏幕震动和 FOV 渐变：

```rust
use anvilkit_camera::effects::CameraEffects;

let mut fx = CameraEffects::default();
fx.head_bob_enabled = true;
fx.add_shake(0.2);          // 落地冲击
fx.fov_target = 10.0;       // 冲刺 FOV 偏移（平滑插值）
```

## 万向锁保护

第三人称 look-at 自动处理万向锁问题——当相机接近目标正上方或正下方时，使用替代 up 向量避免翻转。
