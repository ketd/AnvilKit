`anvilkit-camera` 提供灵活的摄像机控制系统。

## CameraController

```rust
use anvilkit_camera::controller::CameraController;

let controller = CameraController {
    mode: CameraMode::FirstPerson,
    move_speed: 10.0,
    mouse_sensitivity: 0.003,
    base_fov: 70.0, // 基础视野角度（度）
    ..Default::default()
};
```

内置的 `camera_controller_system` 从 `InputState::mouse_delta()` 读取鼠标增量，引擎会自动从 `DeviceEvent::MouseMotion` 填充该值。无需手动管理鼠标增量。

## 摄像机模式

| 模式 | 说明 |
|------|-------------|
| `FirstPerson` | 第一人称 — 摄像机跟随玩家视角 |
| `ThirdPerson` | 第三人称 — 摄像机跟随目标并带有偏移 |
| `Free` | 自由飞行 — 无约束的六自由度运动 |

## 摄像机效果

`CameraEffects` 提供头部摆动、屏幕震动和 FOV 过渡效果：

```rust
use anvilkit_camera::effects::CameraEffects;

let mut fx = CameraEffects::default();
fx.head_bob_enabled = true;
fx.add_shake(0.2);          // 着陆冲击
fx.fov_target = 10.0;       // 冲刺 FOV 偏移（平滑插值）
```

## 万向节锁保护

第三人称注视（look-at）会自动处理万向节锁——当摄像机接近目标正上方或正下方时，使用替代的上方向向量以防止翻转。
