## 概述

`FrameProfiler` 提供 CPU 端的帧计时和游戏循环分段性能分析功能。该功能通过 `"debug"` feature flag 进行门控。当未启用该 feature 时，所有方法编译为空操作，零运行时开销。

该分析器由 `anvilkit-ecs` crate 提供：

```rust
use anvilkit_ecs::profiler::FrameProfiler;
```

`FrameProfiler` 是一个 ECS `Resource`，维护帧时间的滚动历史窗口（默认：300 帧）。它还追踪命名分段的计时数据，并支持外部 GPU 计时数据。

## FrameProfiler API

| 方法 | 签名 | 说明 |
|--------|-----------|-------------|
| `new` | `() -> Self` | 创建新的分析器，使用默认设置（300 帧历史） |
| `begin_frame` | `(&mut self)` | 记录新帧的开始 |
| `end_frame` | `(&mut self)` | 记录当前帧的结束；计算耗时并推入历史记录 |
| `begin_section` | `(&mut self, name: &str)` | 在当前帧中开始一个命名计时分段 |
| `end_section` | `(&mut self, name: &str)` | 结束一个命名计时分段；将耗时存入分段历史 |
| `fps` | `(&self) -> f64` | 历史窗口内的平均每秒帧数 |
| `avg_frame_time_ms` | `(&self) -> f64` | 历史窗口内的平均帧时间（毫秒） |
| `avg_section_time_ms` | `(&self, name: &str) -> Option<f64>` | 命名分段的平均时间（毫秒），无数据时返回 `None` |
| `min_frame_time_ms` | `(&self) -> f64` | 历史窗口内的最小帧时间（毫秒） |
| `max_frame_time_ms` | `(&self) -> f64` | 历史窗口内的最大帧时间（毫秒） |
| `percentile_frame_time_ms` | `(&self, p: f64) -> f64` | 百分位帧时间（毫秒）（例如 `p = 0.99` 表示第 99 百分位） |

### 公开字段（仅 debug feature）

| 字段 | 类型 | 说明 |
|-------|------|-------------|
| `frame_times` | `VecDeque<f64>` | 帧时间滚动历史（秒） |
| `gpu_times` | `VecDeque<f64>` | GPU 时间滚动历史（秒）（由外部填充） |
| `section_times` | `HashMap<String, VecDeque<f64>>` | 命名分段的计时历史（秒） |
| `frame_count` | `u64` | 已记录的总帧数 |
| `max_history` | `usize` | 历史中保留的最大帧数（默认：300） |

## 用法示例

```rust
use anvilkit_ecs::profiler::FrameProfiler;

let mut profiler = FrameProfiler::new();

// 在游戏循环中：
profiler.begin_frame();

profiler.begin_section("physics");
// ... 物理模拟 ...
profiler.end_section("physics");

profiler.begin_section("render");
// ... 渲染 ...
profiler.end_section("render");

profiler.end_frame();

// 查询统计数据：
println!("FPS: {:.1}", profiler.fps());
println!("Avg frame: {:.2} ms", profiler.avg_frame_time_ms());
println!("Min frame: {:.2} ms", profiler.min_frame_time_ms());
println!("Max frame: {:.2} ms", profiler.max_frame_time_ms());
println!("99th percentile: {:.2} ms", profiler.percentile_frame_time_ms(0.99));

if let Some(physics_ms) = profiler.avg_section_time_ms("physics") {
    println!("Physics avg: {:.2} ms", physics_ms);
}
```

当 `"debug"` feature 被禁用时，以上所有调用均编译为空操作。`fps()`、`avg_frame_time_ms()`、`min_frame_time_ms()`、`max_frame_time_ms()` 和 `percentile_frame_time_ms()` 返回 `0.0`，`avg_section_time_ms()` 返回 `None`。
