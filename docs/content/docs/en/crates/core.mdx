`anvilkit-core` is the engine's foundation type library, providing shared infrastructure -- math types, time tracking, and persistence -- for all other crates.

## Transform

`Transform` represents position, rotation, and scale in 3D space (40 bytes). Built on `glam`.

### Fields

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `translation` | `Vec3` | `Vec3::ZERO` | Position in space |
| `rotation` | `Quat` | `Quat::IDENTITY` | Orientation as a unit quaternion |
| `scale` | `Vec3` | `Vec3::ONE` | Per-axis scale factors |

### Constructors

| Method | Signature | Description |
|--------|-----------|-------------|
| `IDENTITY` | `const` | Zero translation, identity rotation, unit scale |
| `new` | `(Vec3, Quat, Vec3) -> Self` | Full constructor |
| `from_translation` | `(Vec3) -> Self` | Position only |
| `from_rotation` | `(Quat) -> Self` | Rotation only |
| `from_scale` | `(Vec3) -> Self` | Scale only |
| `from_xyz` | `(f32, f32, f32) -> Self` | Position from x/y/z |
| `from_xy` | `(f32, f32) -> Self` | 2D position (z = 0) |
| `from_matrix` | `(Mat4) -> Self` | Decompose a 4x4 matrix |
| `looking_at` | `(eye: Vec3, target: Vec3, up: Vec3) -> Result<Self>` | Camera-style orientation |

### Builder Methods (chainable)

| Method | Signature |
|--------|-----------|
| `with_translation` | `(self, Vec3) -> Self` |
| `with_rotation` | `(self, Quat) -> Self` |
| `with_scale` | `(self, Vec3) -> Self` |

### Operations

| Method | Signature | Description |
|--------|-----------|-------------|
| `compute_matrix` | `(&self) -> Mat4` | Build the Scale-Rotation-Translation matrix |
| `transform_point` | `(&self, Vec3) -> Vec3` | Apply full transform to a point |
| `transform_vector` | `(&self, Vec3) -> Vec3` | Apply rotation+scale only (ignores translation) |
| `mul_transform` | `(&self, &Transform) -> Transform` | Compose `self * other` |
| `inverse` | `(&self) -> Result<Transform>` | Compute inverse (fails if any scale axis is zero) |
| `is_finite` | `(&self) -> bool` | True if no NaN/Inf in any field |

```rust
use anvilkit_core::math::Transform;
use glam::{Vec3, Quat};

// Create and chain
let t = Transform::from_xyz(1.0, 2.0, 3.0)
    .with_rotation(Quat::from_rotation_y(std::f32::consts::FRAC_PI_4))
    .with_scale(Vec3::splat(2.0));

// Compose parent * child
let parent = Transform::from_xyz(10.0, 0.0, 0.0);
let child  = Transform::from_xyz(5.0, 0.0, 0.0);
let world  = parent.mul_transform(&child);
assert!((world.translation - Vec3::new(15.0, 0.0, 0.0)).length() < 1e-5);

// Inverse
let inv = t.inverse().unwrap();
let identity = t.mul_transform(&inv);
assert!(identity.translation.length() < 1e-4);
```

## GlobalTransform

`GlobalTransform` wraps a `Mat4` and represents the final world-space transform.
It is typically computed automatically by the transform propagation systems in `anvilkit-ecs`.

### API

| Method | Signature | Description |
|--------|-----------|-------------|
| `IDENTITY` | `const` | Identity matrix |
| `from_matrix` | `(Mat4) -> Self` | Wrap an existing matrix |
| `from_transform` | `(&Transform) -> Self` | Convert from a local Transform |
| `matrix` | `(&self) -> Mat4` | Get the raw matrix |
| `translation` | `(&self) -> Vec3` | Extract the translation column |
| `rotation` | `(&self) -> Quat` | Decompose and return rotation |
| `scale` | `(&self) -> Vec3` | Decompose and return scale |
| `transform_point` | `(&self, Vec3) -> Vec3` | Transform a point |
| `transform_vector` | `(&self, Vec3) -> Vec3` | Transform a vector (no translation) |
| `mul_transform` | `(&self, &GlobalTransform) -> GlobalTransform` | Matrix multiplication |
| `inverse` | `(&self) -> Result<Self>` | Matrix inverse (fails if singular) |
| `is_finite` | `(&self) -> bool` | True if all elements are finite |

`GlobalTransform` implements `From<Transform>` and `From<Mat4>`.

```rust
use anvilkit_core::math::{Transform, GlobalTransform};
use glam::Vec3;

let local = Transform::from_xyz(5.0, 10.0, 15.0);
let global: GlobalTransform = local.into();
assert_eq!(global.translation(), Vec3::new(5.0, 10.0, 15.0));
```

## Time

`Time` tracks frame delta, total elapsed time, and frame count. It is usually stored as an ECS resource and updated once per frame.

### API

| Method | Signature | Description |
|--------|-----------|-------------|
| `new` | `() -> Self` | Create with current instant as startup time |
| `update` | `(&mut self)` | Advance to current instant (call once per frame) |
| `delta` | `(&self) -> Duration` | Time since last `update()` |
| `delta_seconds` | `(&self) -> f32` | Delta as `f32` seconds |
| `delta_seconds_f64` | `(&self) -> f64` | Delta as `f64` seconds |
| `delta_millis` | `(&self) -> u128` | Delta as milliseconds |
| `elapsed` | `(&self) -> Duration` | Time since startup |
| `elapsed_seconds` | `(&self) -> f32` | Elapsed as `f32` seconds |
| `frame_count` | `(&self) -> u64` | Total frames processed |
| `fps` | `(&self) -> f64` | Average FPS (frames / elapsed) |
| `instant_fps` | `(&self) -> f64` | Instantaneous FPS (1 / delta) |
| `is_first_frame` | `(&self) -> bool` | True before the first `update()` |
| `reset` | `(&mut self)` | Reset all counters to zero |
| `with_scale` | `(&self, f32) -> ScaledTime` | Create a time-scaled wrapper |

```rust
use anvilkit_core::time::Time;

let mut time = Time::new();

// Game loop
time.update();
let dt = time.delta_seconds();   // seconds since last update
let total = time.elapsed_seconds();
println!("Frame {} | dt={:.4}s | FPS={:.1}", time.frame_count(), dt, time.fps());

// Slow motion
let slow = time.with_scale(0.5);
let slow_dt = slow.delta_seconds(); // dt * 0.5
```

## Timer

`Timer` provides one-shot and repeating countdown timers with pause/resume support.

### Constructors

| Method | Description |
|--------|-------------|
| `new(Duration)` | One-shot timer |
| `repeating(Duration)` | Repeating timer |
| `from_seconds(f32)` | One-shot from seconds |
| `repeating_from_seconds(f32)` | Repeating from seconds |
| `from_millis(u64)` | One-shot from milliseconds |
| `repeating_from_millis(u64)` | Repeating from milliseconds |

### Core API

| Method | Signature | Description |
|--------|-----------|-------------|
| `tick` | `(&mut self, Duration)` | Advance the timer by delta |
| `finished` | `(&self) -> bool` | True when elapsed >= duration |
| `just_finished` | `(&self) -> bool` | True only on the frame the timer completes |
| `percent` | `(&self) -> f32` | Progress 0.0..1.0 |
| `elapsed` | `(&self) -> Duration` | Time elapsed so far |
| `remaining` | `(&self) -> Duration` | Time left |
| `reset` | `(&mut self)` | Reset elapsed to zero, state to Running |
| `pause` | `(&mut self)` | Pause (tick becomes no-op) |
| `resume` | `(&mut self)` | Resume from paused state |
| `finish` | `(&mut self)` | Force-complete immediately |
| `set_duration` | `(&mut self, Duration)` | Change the target duration |
| `set_repeating` | `(&mut self, bool)` | Toggle repeating mode |
| `is_running` / `is_paused` | `(&self) -> bool` | State queries |

```rust
use anvilkit_core::time::Timer;
use std::time::Duration;

// Weapon cooldown: 0.5s repeating
let mut cooldown = Timer::repeating_from_seconds(0.5);

// Each frame
cooldown.tick(Duration::from_millis(16));
if cooldown.just_finished() {
    // Can fire again
}

// Delayed event: 3 seconds one-shot
let mut delay = Timer::from_seconds(3.0);
delay.tick(delta);
if delay.just_finished() {
    // Trigger cutscene
}
```

## Persistence (feature `"persistence"`)

The persistence module provides three layers: player **Settings**, multi-slot **SaveManager**, and raw key-value **WorldStorage**.

### Settings

Typed configuration stored as RON. Falls back to defaults when the file is missing or corrupt.

#### Sub-structs

**GraphicsSettings**

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `width` | `u32` | `1280` | Window width |
| `height` | `u32` | `720` | Window height |
| `fullscreen` | `bool` | `false` | Fullscreen mode |
| `vsync` | `bool` | `true` | Vertical sync |
| `msaa` | `u32` | `4` | MSAA samples (1/2/4) |
| `bloom` | `bool` | `true` | Bloom post-process |
| `ssao` | `bool` | `true` | Screen-space AO |
| `shadow_quality` | `u32` | `2` | 0=off, 1=low, 2=med, 3=high |

**AudioSettings**

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `master_volume` | `f32` | `1.0` | Master volume 0.0..1.0 |
| `music_volume` | `f32` | `0.8` | Music volume 0.0..1.0 |
| `sfx_volume` | `f32` | `1.0` | SFX volume 0.0..1.0 |

**InputSettings**

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `mouse_sensitivity` | `f32` | `0.003` | Mouse sensitivity multiplier |
| `invert_y` | `bool` | `false` | Invert Y axis |
| `action_overrides` | `HashMap<String, String>` | `{}` | Custom key rebindings |

#### Settings API

| Method | Signature | Description |
|--------|-----------|-------------|
| `load` | `(path: &Path) -> Self` | Load from RON; returns defaults on error |
| `save` | `(&self, path: &Path) -> Result<(), String>` | Write to RON (creates dirs) |
| `default_path` | `() -> PathBuf` | `"config/settings.ron"` |

```rust
use anvilkit_core::persistence::Settings;

// Load (or default)
let mut settings = Settings::load(&Settings::default_path());
settings.graphics.vsync = false;
settings.audio.master_volume = 0.7;
settings.save(&Settings::default_path()).unwrap();
```

### SaveManager

Manages multiple save slots, each with metadata (`meta.ron`) and a `data/` directory for `WorldStorage`.

#### Directory Layout

```
saves/
  quick/
    meta.ron
    data/        <-- WorldStorage root
  slot_1/
    meta.ron
    data/
```

#### SaveSlotInfo Fields

| Field | Type | Description |
|-------|------|-------------|
| `name` | `String` | Slot name (e.g. `"quick"`, `"slot_1"`) |
| `timestamp` | `u64` | Unix timestamp (seconds) |
| `play_time_secs` | `f64` | Cumulative play time |
| `game_version` | `String` | Game version string |
| `metadata` | `HashMap<String, String>` | Game-specific metadata |

#### SaveManager API

| Method | Signature | Description |
|--------|-----------|-------------|
| `new` | `(dir: impl AsRef<Path>, version: &str) -> Result<Self, String>` | Create/open saves directory |
| `save` | `(&self, slot: &str, play_time: f64, metadata: HashMap) -> Result<PathBuf, String>` | Write metadata, return `data/` path |
| `list_saves` | `(&self) -> Vec<SaveSlotInfo>` | All slots sorted newest-first |
| `get_save_info` | `(&self, slot: &str) -> Option<SaveSlotInfo>` | Read one slot's metadata |
| `delete` | `(&self, slot: &str) -> Result<(), String>` | Remove slot directory recursively |
| `slot_data_path` | `(&self, slot: &str) -> PathBuf` | Get `data/` path without writing metadata |

```rust
use anvilkit_core::persistence::SaveManager;
use std::collections::HashMap;

let mgr = SaveManager::new("saves", "1.0.0").unwrap();

// Create a save
let mut meta = HashMap::new();
meta.insert("level".into(), "3".into());
let data_path = mgr.save("slot_1", 3600.0, meta).unwrap();

// List saves
for info in mgr.list_saves() {
    println!("{}: played {}s", info.name, info.play_time_secs);
}

// Delete
mgr.delete("slot_1").unwrap();
```

### WorldStorage

File-system key-value store. Each key maps to a file under the base directory. Slashes in keys create subdirectories. Writes are atomic (write to `.tmp` then rename).

#### API

| Method | Signature | Description |
|--------|-----------|-------------|
| `open` | `(path: impl AsRef<Path>) -> Result<Self, String>` | Open/create the storage directory |
| `get` | `(&self, key: &str) -> Option<Vec<u8>>` | Read value; `None` if missing |
| `put` | `(&self, key: &str, value: &[u8]) -> Result<(), String>` | Write atomically (tmp + rename) |
| `delete` | `(&self, key: &str) -> Result<(), String>` | Remove a key |
| `keys_with_prefix` | `(&self, prefix: &str) -> Vec<String>` | List keys starting with prefix |
| `batch_put` | `(&self, entries: &[(&str, &[u8])]) -> Result<(), String>` | Write multiple keys (each atomic) |
| `path` | `(&self) -> &Path` | Base directory |

```rust
use anvilkit_core::persistence::WorldStorage;

let storage = WorldStorage::open("saves/slot_1/data").unwrap();

// Store chunk data
storage.put("chunk/0/0", &bincode::serialize(&chunk).unwrap()).unwrap();

// Read it back
if let Some(data) = storage.get("chunk/0/0") {
    let chunk: Chunk = bincode::deserialize(&data).unwrap();
}

// Enumerate all chunks
let keys = storage.keys_with_prefix("chunk/");
println!("Stored {} chunks", keys.len());

// Batch write
storage.batch_put(&[
    ("meta/version", b"1"),
    ("meta/seed", b"42"),
]).unwrap();
```
