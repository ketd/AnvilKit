## UI System

Flexbox-based UI with GPU-accelerated rectangle rendering, event system, and widget constructors.

**Layout**: Powered by `taffy` (full CSS Flexbox subset). `UiLayoutEngine::compute_layout()` takes `UiNode` entities and a container size, outputs `[x, y, w, h]` computed rectangles.

**Rendering**: `UiRenderer` draws rounded rectangles with background color, border, and text in a single draw call with alpha blending over the 3D scene.

**Event System**:

| Event | Trigger |
|-------|---------|
| `HoverEnter` | Mouse enters node bounds |
| `HoverLeave` | Mouse leaves node bounds |
| `Click` | Mouse button pressed within node |

```rust
use anvilkit_render::renderer::ui::{UiEvents, Widget, UiNode};

// Create widgets
let button = Widget::button("Start Game");
let panel = Widget::panel();
let label = Widget::label("Score: 100");
let row = Widget::row();    // horizontal layout
let col = Widget::column();  // vertical layout

// Check events
let events = world.resource::<UiEvents>();
if events.was_clicked(button_entity) {
    // Handle button click
}
if events.is_hovered(button_entity) {
    // Show hover effect
}
```

**UiStyle fields**: `flex_direction`, `justify_content`, `align_items`, `width`, `height`, `min_width`, `min_height`, `max_width`, `max_height`, `padding[4]`, `margin[4]`, `gap`, `flex_grow`, `flex_shrink`.

## Custom Render Pipeline

For games that bypass the built-in PBR pipeline (e.g., voxel games), use `RenderApp` in low-level mode with the helper API:

```rust
use anvilkit_render::window::RenderApp;

impl ApplicationHandler for MyGame {
    fn window_event(&mut self, el, wid, ev) {
        // Game-specific key handling here...

        // Engine handles input forwarding (keyboard, mouse, cursor, scroll)
        RenderApp::forward_input(&mut self.app, &ev);
        self.render_app.window_event(el, wid, ev);
    }

    fn device_event(&mut self, el, did, ev) {
        // Engine forwards raw mouse motion → InputState.mouse_delta
        RenderApp::forward_device_input(&mut self.app, &ev);
        self.render_app.device_event(el, did, ev);
    }

    fn about_to_wait(&mut self, _el) {
        // Engine tick: DeltaTime → app.update() → InputState.end_frame() → request_redraw
        self.render_app.tick(&mut self.app);
        // Your post-update logic here...
    }
}
```
