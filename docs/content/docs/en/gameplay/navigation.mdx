The navigation module provides mesh-based pathfinding using A* on a triangle adjacency graph, plus an agent steering system for automated movement.

## NavMesh

`NavMesh` is an ECS `Resource` that stores the walkable surface as triangles with auto-computed adjacency.

### Construction

```rust
use anvilkit_ecs::navigation::NavMesh;
use glam::Vec3;

let vertices = vec![
    Vec3::new(0.0, 0.0, 0.0),
    Vec3::new(10.0, 0.0, 0.0),
    Vec3::new(10.0, 0.0, 10.0),
    Vec3::new(0.0, 0.0, 10.0),
];
let triangles = vec![
    [0, 1, 2],
    [0, 2, 3],
];

// Adjacency is computed automatically from shared edges
let navmesh = NavMesh::new(vertices, triangles);
```

> **Performance:** `build_adjacency` runs in **O(n)** time using an edge hash map, where *n* is the number of triangles.

### API

| Method | Signature | Description |
|--------|-----------|-------------|
| `new` | `(Vec<Vec3>, Vec<[u32; 3]>) -> Self` | Build NavMesh with auto-adjacency |
| `find_triangle` | `(&self, Vec3) -> Option<usize>` | Find which triangle contains a point (XZ projection) |
| `triangle_centroid` | `(&self, usize) -> Vec3` | Get the centroid of a triangle |
| `find_path` | `(&self, Vec3, Vec3) -> Option<Vec<Vec3>>` | A* pathfinding, returns waypoints |
| `point_in_triangle` | `(Vec2, Vec2, Vec2, Vec2) -> bool` | Barycentric point-in-triangle test |

> `point_in_triangle` safely handles degenerate triangles (zero area) by returning `false` instead of dividing by zero.

## A* Pathfinding

`find_path(start, goal)` runs A* on the triangle adjacency graph using centroids as nodes and Euclidean distance as the heuristic. Returns a list of waypoints: `[start, centroid_1, centroid_2, ..., goal]`.

```rust
if let Some(path) = navmesh.find_path(
    Vec3::new(1.0, 0.0, 1.0),  // start
    Vec3::new(8.0, 0.0, 8.0),  // goal
) {
    for waypoint in &path {
        println!("  -> {}", waypoint);
    }
}
```

## NavAgent

`NavAgent` is a `Component` that moves an entity along a computed path each frame.

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `speed` | `f32` | — | Movement speed (units/sec) |
| `path` | `Vec<Vec3>` | `[]` | Current waypoint path |
| `current_waypoint` | `usize` | `0` | Index into path |
| `arrival_threshold` | `f32` | `0.3` | Distance to consider waypoint reached |
| `enabled` | `bool` | `true` | Whether the agent is active |

| Method | Signature | Description |
|--------|-----------|-------------|
| `new` | `(speed: f32) -> Self` | Create a new agent |
| `set_destination` | `(&mut self, &NavMesh, Vec3, Vec3) -> bool` | Compute path, returns false if no path |
| `has_arrived` | `(&self) -> bool` | True if at the end of the path |

```rust
use anvilkit_ecs::navigation::{NavAgent, NavMesh};

// Spawn an NPC with a NavAgent
let mut agent = NavAgent::new(5.0); // 5 units/sec
agent.set_destination(&navmesh, npc_pos, target_pos);
commands.spawn((Transform::from_translation(npc_pos), agent));
```

## NavigationPlugin

`NavigationPlugin` registers `nav_agent_steering_system` in `Update`. The system moves all `NavAgent` entities along their paths using `DeltaTime`.

```rust
app.add_plugins(NavigationPlugin);
app.insert_resource(navmesh);
```
